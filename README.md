## Описание проекта

В ходе лабораторной работы было создано интерактивное веб-приложение "Калькулятор", выполняющее базовые арифметические операции. Проект реализован с использованием чистого JavaScript, HTML5 и CSS3. Особое внимание уделено архитектуре приложения — применён объектно-ориентированный подход с разделением логики и представления, а также использование ES6 модулей.

## Функциональность

### Поддерживаемые операции:
- Сложение (`+`)
- Вычитание (`-`)
- Умножение (`*`)
- Деление (`/`)
- Десятичные дроби (`.`)
- Очистка поля ввода (`C`)
- Вычисление результата (`=`)

### Обработка ошибок:
- Запрет ввода двух операторов подряд (например, `5++3`)
- Запрет оператора в начале выражения (кроме минуса для отрицательных чисел)
- Запрет нескольких точек в одном числе (например, `5.2.1`)
- Перехват исключений при вычислении (`eval`)
- Отображение сообщения "Ошибка" при некорректном выражении
- Защита от деления на ноль

### Особенности:
- Автоматический сброс состояния после ошибки
- Возможность продолжать вычисления с полученным результатом
- Интуитивно понятный интерфейс

### Класс Calculator

Класс инкапсулирует всю логику работы калькулятора:

| Метод | Назначение |
|-------|------------|
| `constructor()` | Инициализация DOM-элементов и состояния |

## Логика работы

### Последовательность действий при нажатии кнопки

1. Пользователь нажимает кнопку
2. Срабатывает обработчик события в `init()`
3. Вызывается `handleButtonClick(value)`
4. В зависимости от значения:
   - `"C"` → `clear()`
   - `"="` → `calculate()`
   - иначе → `addToExpression(value)`
5. Обновляется дисплей через `updateDisplay()`

   if (this.isOperator(value) && this.isLastCharOperator()) {
       this.expression = this.expression.slice(0, -1) + value;

Заменяет последний оператор новым вместо добавления.
